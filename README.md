# MoyaTodos
Simple todos app using [Moya](https://github.com/Moya/Moya) for demo purposes.


## Getting Started

### 1. Download dependencies
```bash
pod install
```

### 2. Run in Xcode
Open **MoyaTodos.xcworkspace** and run to your simulator or iPhone.


## Requirements

- iOS 9+
- macOS with Xcode 9.3+ 
- Swift 4+
